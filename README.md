# SmartSpend 💰

SmartSpend is a MERN stack personal finance management system.

## Features
- Track expenses
- Budget management
- Data visualization
- OCR receipt scanning
- Smart categorization

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB

## How to Run

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```
